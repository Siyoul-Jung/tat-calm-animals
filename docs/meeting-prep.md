## ❓ Tapas에게 확인할 것 (4가지)

### 1. 디자인 방향 — "이 느낌이 맞나요?"

현재 사이트의 전체 비주얼 방향을 함께 검토합니다.

- 색상: 다크 그린(`#1F2E14`) + 오렌지(`#D4703A`) 조합
- 폰트: Playfair Display(헤딩) + DM Sans(본문)
- 톤: 따뜻하고 안정적 — 영적이지만 압도적이지 않은 느낌

**→ 보여줄 것:** 현재 사이트 라이브 데모

---

### 2. 폰트 선택 — Sans-serif 본문 폰트 최종 결정

현재 DM Sans를 쓰고 있으나, 다른 옵션도 준비했습니다.

**→ 보여줄 것:** `/font-preview` 페이지 — 같은 문장을 3가지 폰트로 비교

---

### 3. 홈페이지 영상 — 어떤 영상을 쓸까?

YouTube 채널을 검토한 결과, **TAT for Animals 전용 공개 영상이 없습니다.**

두 가지 선택지가 있습니다:

| 옵션 | 방법 | 시간/비용 |
|------|------|---------|
| A | 기존 비디오 중 홈페이지에 공개 가능한 것 사용 | 즉시 가능 |
| B | 짧은 소개 영상 신규 촬영 (2–3분) | 촬영 일정 필요 |

**→ Tapas + Jez 결정 필요**
