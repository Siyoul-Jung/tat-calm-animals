TAT for Animals — Project Summary

tatforanimals.com  ·  Prepared by Siyoul  ·  April 26, 2026

Overview

tatforanimals.com is a new, independent membership platform built specifically for animal owners. It is completely separate from tatlife.com — nothing on the existing site changes.

The site is designed to feel calm, warm, and easy to use. A visitor's first impression should be felt before it is understood — the design puts experience first, information second.

Content Management

Jez can manage all content independently — no developer needed for day-to-day updates.

For design changes, new features, or structural updates to the site, a developer with Next.js experience will be needed. Any experienced Next.js developer can get up to speed with this codebase in 1–2 days.

TaskHowTime
