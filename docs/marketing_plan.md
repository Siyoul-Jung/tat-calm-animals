# TAT for Animals — Marketing Plan

**기준:** 런칭 전 3개월 + 런칭 후 3개월  
*Last updated: April 2026*

---

## 1. 타깃 고객 분석

**핵심 타깃:**
반려동물이 뭔가를 안고 있다고 느끼는 보호자. 수의사 진단으로 해결되지 않는 불안, 트라우마, 행동 변화를 경험하고 있고, 자연적이고 온화한 방법을 찾고 있는 사람.

| 세그먼트 | 특징 | 접근 방식 |
|---------|------|---------|
| **코어** | 기존 TAT/TATLife 구독자 | 직접 전환 — 이미 신뢰 있음 |
| **확장** | 동물 치유/홀리스틱 케어 관심층 | 콘텐츠로 교육 후 전환 |
| **신규** | 반려동물 불안 문제 검색 중인 보호자 | SEO + YouTube 검색 유입 |

---

## 2. 핵심 메시지

> "당신의 동물은 이미 당신과 연결되어 있어요. TAT는 그 연결을 치유로 바꿉니다."

**피해야 할 메시지:**
- "과학적으로 검증된" — 증명이 어렵고 타깃과 맞지 않음
- "세계 80개국" — 숫자보다 이야기가 설득력 있음
- "혁신적인 기술" — 차갑고 거리감 있음

---

## 3. 마케팅 채널 전략

### 🥇 1순위 — 기존 구독자 이메일
이미 Tapas를 신뢰하는 사람들. 전환율이 소셜보다 압도적으로 높음.

- 기존 TATLife 구독자에게 얼리버드 안내
- 런칭 전 웨이팅 리스트 구축 (무료 입문 영상 제공 조건)
- 월 1-2회 뉴스레터: 치유 이야기 + 웨비나 안내
- 자동화: 웨비나 초대, 녹화 알림, 구독 갱신 리마인더

### 🥈 2순위 — YouTube (TATLife 채널)
기존 채널 이미 운영 중. 동물 관련 영상 존재. 마지막 업로드 5-6개월 전 — 재활성화 필요.

**왜 YouTube가 중요한가:**
- 검색 의도 기반 유입 ("개 불안 자연치료", "고양이 트라우마 치유")
- 영상 하나가 수년간 유입 지속 — 장기 자산
- 40-60대 타깃에게 신뢰도 높은 플랫폼
- 긴 영상 = 깊은 신뢰 형성

**콘텐츠 방향:**
- 월 2-4회 업로드 목표 (꾸준함이 핵심)
- 동물별 케이스: "불안한 개에게 TAT 하는 법", "고양이 트라우마"
- 실제 세션 영상 — 동물이 반응하는 순간이 최고의 콘텐츠
- Tapas의 홈페이지 소개 영상 (3-4분) → YouTube에도 공개

### 🥉 3순위 — 팟캐스트 게스트 출연
광고비 없이 신뢰도 높은 오디언스에 도달하는 가장 효율적인 방법.

- 반려동물, 홀리스틱 케어, 동물 심리 관련 팟캐스트 타겟
- Tapas의 스토리 + TAT for Animals 소개
- 에피소드 당 수백~수천 명의 따뜻한 오디언스에 노출

### 4순위 — Instagram
YouTube 영상을 30-60초 Reels로 재활용. 별도 제작 부담 최소화.

- Reels: TAT 시연 클립, 동물 반응 순간
- Stories: Tapas 일상 + 구독자 후기
- 주 3회 목표

### 5순위 — Facebook
유기적 도달 낮음. 광고 예산 투입 시점에 활용.

- 페이지 운영보다 **그룹 참여** 중심
- 반려동물 보호자 그룹에 Tapas 콘텐츠 공유
- 유료 광고: 오가닉 효과 검증 후 투입

## 4. 런칭 전략 (3단계)

### Phase 1 — 씨앗 심기 (런칭 6주 전)
- 기존 TATLife 구독자에게 얼리버드 안내 이메일
- "Coming Soon" 페이지 + 웨이팅 리스트 수집
- YouTube 재활성화 — 동물 콘텐츠 업로드 재개
- Instagram 티저 콘텐츠 시작 (주 3회)

### Phase 2 — 열기 만들기 (런칭 2주 전)
- 웨이팅 리스트에 얼리버드 혜택 제공
- 기존 구독자 후기 2-3개 소셜에 공개
- Tapas 라이브 Q&A 세션 1회 — 무료 공개

### Phase 3 — 런칭
- 런칭 당일 이메일 + 소셜 동시 발송
- 첫 2주: 매일 스토리 업데이트
- 첫 번째 라이브 웨비나 (The Calm Circle 가입 유도)

## 5. 전환 최적화

**무료 → 유료 전환 핵심 레버:**
- 홈페이지 무료 영상 → 대시보드 미리보기 → 가입 유도
- "$1 첫 달 체험" 또는 "첫 달 무료" 검토 (Tapas 확인 필요)
- 후기는 텍스트가 아닌 **구체적인 이야기**로 — "루나가 산책을 다시 즐기게 됐어요"

**The Calm Library → The Calm Circle 업그레이드 유도:**
- 웨비나 당일 리마인더 이메일
- 대시보드에 "다음 웨비나: X일 후" 배너
